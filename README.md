# AI Agent Mastery Training

This repository contains the code and learning materials for the AI Agent Mastery training program.

## Week 1 - Agent Fundamentals and Setup

This week focuses on:
- Agent Fundamentals: Chatbots vs. Agents, Levels of Autonomy, Agent Loop.
- Environment Setup: Google Colab, LangGraph, Google ADK, Arize Phoenix.
- LLM API Integration: OpenAI, Gemini.
- Basic Tool Integration.
- State Management & Memory.
- Structured Output & Security Mindset.
- Hands-On Project: Topic Research Agent (LangGraph & ADK).

## Setup
Follow the instructions in the notebook to set up your Google Colab environment and configure API keys.
